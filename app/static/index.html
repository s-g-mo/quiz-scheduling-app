Tips for Generating Schedules

  1. Struggling to generate a valid schedule? Try setting the number of matches per team to 3 and the number of teams to 7 or more. This setup tends to give the best results. See the documentation for more details.
  2. Still no luck? Try increasing the number of time slots first, then adjust the number of rooms if needed.
